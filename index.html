Scrypted

Scrypted is a home automation platform powered by Android.

Use your favorite app

Scrypted integrates with the three major home hubs. Control your things with via voice or your preferred app, whether it is Google Home, Apple HomeKit, or Amazon Alexa.

Automate

  • Schedules

    Set up daily and weekly schedules to automate your things.

  • Events

    React to events from switches, sensors, and more.

  • Control

    Perform common actions with the built in wizard, or write a custom script for finer control.

Developers

Scrypted can be extended with JavaScript plugins: write scripts that control your things or add support for entirely new devices.

  • VS Code
  •           JavaScript
  • Debug

Simple scripts can be written in the management console.
Advanced plugins can be written with Visual Studio Code. Use popular npm modules, deploy directly to your Scrypted server, and debug in a full fledged development environment.

  • View Documentation

Download

Scrypted is an Android app that can be run on any Android Lollipop (5.0+) device.
Support for Aeon Labs Z-Stick Gen 2/5 sticks has been tested with NVidia Shield, Fire TV, and a Raspberry Pi (Android Things.)
